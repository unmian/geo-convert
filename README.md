# @quarter/geo-convert

> 地理空间坐标系转换工具

## 安装

配置私有仓库

```txt
// .yarnrc
"@quarter:registry" "https://npm.lescity.com.cn/"

//.npmrc
@quarter:registry=https://npm.lescity.com.cn/
```

安装依赖

```bash
npm install @quarter/geo-convert
# 或者
yarn add @quarter/geo-convert
```
