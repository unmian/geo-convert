# @unmian/geo-convert

> 地理空间坐标系转换工具

## 安装

配置私有仓库

```txt
// .yarnrc
"@unmian:registry" "https://npm.lescity.com.cn/"

//.npmrc
@unmian:registry=https://npm.lescity.com.cn/
```

安装依赖

```bash
npm install @unmian/geo-convert
# 或者
yarn add @unmian/geo-convert
# 或者
pnpm install @unmian/geo-convert
```
